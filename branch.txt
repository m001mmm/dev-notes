# 1. 创建并切换到新分支（开发环境）
git checkout -b feature/xxx

# 2. 编写代码（你自己的操作）

# 3. 添加改动到暂存区
git add .

# 4. 提交代码（生成版本记录）
git commit -m "xxx"

# 5. （可选）把功能分支推送到 GitHub
git push origin feature/xxx

# 6. 切换回主分支
git checkout main

# 7. 合并功能分支到 main
git merge feature/xxx

# 8. 推送 main 到 GitHub（必须）
git push

| 步骤            | 作用             |
| ------------- | -------------- |
| checkout -b   | 开新分支开发         |
| add           | 选中要提交的代码       |
| commit        | 保存一个版本         |
| push feature  | 上传功能分支（可选）     |
| checkout main | 回主分支           |
| merge         | 把功能合并进 main    |
| push          | 同步到 GitHub（必须） |
